属性：resizeMode 当图片尺寸与组件的尺寸不一致时，如何调整图片大小
属性值：
cover（默认）：图片等比例缩放，直到图片宽高都大于等于容器的宽高， 超出容器宽高的部分被  隐藏（减去 padding）
本地图片且未设置容器（Image 标签）的宽高属性，按照原图显示，否则按照容器宽高显示（减去 padding）
网络图片按照 Image 宽高显示

_效果_

contain： 图片等比例缩放,直到图片宽高都小于容器宽高
 可能出现图片未覆盖区域

_效果_

stretch：图片根据 Image 大小进行调整，占满整个标签
图片可能变形

_效果_

repeat：重复平铺图片直到填满容器，当尺寸超过容器时，图片会保持原有尺寸缩放到容器中

_效果_

center：居中显示不拉伸图片

  • 常用方法
    onLoad 加载成功完成时调用此回调函数。
    onLoadEnd 加载结束后，不管成功还是失败，调用此回调函数。
    onLoadStart 图片开始加载时调用
    onLayout 图片加载或布局改变时调用。参数为{nativeEvent:{layout{x,y,width,height}}}

  • 样式属性
    FlexBox：支持弹性布局
    Transforms：支持动画属性
    backgroundColor： 背景颜色
    borderWidth：边框宽度
    borderColor：边框颜色（需要设定宽度后才有颜色）
    borderRadius： 边框圆角
    overflow：设置图片  尺寸超出容器时， 显示或隐藏（visible，hidden(默认））
    tintColor：为所有非透明的像素设置颜色,一旦设置，该  标签  区域内所有内容均被该颜色覆盖
    opacity：透明度
    overlayColor：当图片有圆角时， 指定颜色填充圆角处的空白，
    Android 在 resizeMode 是 contain 或 GIF 动画不支持圆角透明，可任以使用该属性设置圆角颜色为背景颜色
