FlexBox

与PC端的flex基本一致，弹性布局，
不同是，flex的 flexDirection 默认是row，而 FlexBox 默认是column
flexDirection 布局的主轴方向
row:主轴为水平方向，起点在左端
row-reverse:主轴为水平方向，起点在右端
column:主轴为垂直方向，起点在上端
column-reverse:主轴为垂直方向，起点在下端
justifyContent 子元素在主轴线的对齐方式
flex-start（默认）:向主轴起点靠齐
flex-end:向主轴终点靠齐
space-between:两端对齐，第一个和最后一个元素在最边上，两个元素之间距离相等
space-around:元素会平均分布在行内，两端距离父元素边界的距离是两个元素距离的一半
center:居中
alignItems 元素在与主轴垂直的交叉轴上的分部
flex-start：交叉轴轴的起点对齐
flex-end:向交叉轴轴终点靠齐
center：交叉轴的中点对齐
stretch（默认):如果子元素没有设置高度或设置为 auto，将占满整个容器高度
flexWrap 元素在主轴上排列是否可以换行
nowrap(默认)不换行，
wrap:换行

子元素的属性flex
（flex-grow、flex-shrink、flex-basis三个属性的缩写，第二个参数和第三个参数是可选参数）：默认值为 “0 1 auto”
当前元素的flex/所有元素的flex之和是该元素在主轴上占的宽度或高度
如果主轴是column，设置flex后，元素的宽度失效，根据flex计算宽度，
如果设置了高度元素按照高度布局否则撑满父元素的高度
